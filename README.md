# 🎮 Великая Игра Про Кирилла

Текстовая RPG на Python. Твои выборы влияют на карму и концовку!

## 🚀 Как запустить
1. Скачай Python: python.org
2. Запусти: `python new.py`

## 📦 Версии
- v0.1 (beta) ✅ База: выборы, карма, 3 концовки
- v0.2 (beta) ✅ Новые механики: голод, деньги, контакты и рандомные события
